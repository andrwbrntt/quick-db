# Quick DB
Quick DB is a tkinter GUI that allows a user to either import an existing .db file
and query, or convert a .csv to .db and query.


## Goals
The primary goals of this project are to:
- Use basic Python skills cohesively
- Brush up on classes and methods
- Learn how to properly structure an app
- Learn and integrate new concepts


## Tools
While Copilot via VSCode is used as a *guide* in this project, the project itself is a result of myself.

To better aid in legitimate learning, all autocomplete was turned off in VSCode.
